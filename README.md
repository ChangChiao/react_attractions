## 簡介

提供使用者查詢台灣地區的景點、餐廳、活動 
使用React Hook開發﹐搭配styled-component撰寫樣式﹐使用reduxjs/toolki來管理狀態


## 實作功能

- 響應式設計
- 分段加載資料﹐避免一次載入太多筆資料﹐渲染負擔過重
- 依照使用者輸入的條件篩選資料
